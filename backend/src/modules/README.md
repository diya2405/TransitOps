# Backend Module Layout

Paste teammate files into the matching module folder.

- `auth/`: login, register, me, and session-related route/service code
- `shared/`: reusable helpers, middleware adapters, validation, and constants
- `fleet-manager/`: fleet manager-only routes and services
- `driver/`: driver-related backend routes and services
- `safety-officer/`: safety and compliance routes/services
- `financial-analyst/`: finance and reporting routes/services

Suggested subfolders inside each module folder:
- `routes/`
- `services/`
- `utils/`
- `validators/`
