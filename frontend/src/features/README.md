# Frontend Feature Layout

Paste teammate files into the matching role folder.

- `auth/`: login, registration, password/session UI
- `shared/`: shared components, hooks, helpers, and UI primitives
- `fleet-manager/`: fleet manager dashboard and management screens
- `driver/`: driver dashboard and trip screens
- `safety-officer/`: safety officer dashboard, driver monitoring, trip cancellation UI
- `financial-analyst/`: reports, expense, and finance screens

Suggested subfolders inside each role folder:
- `pages/`
- `components/`
- `hooks/`
- `utils/`
